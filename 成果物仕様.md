# 成果物仕様

このファイルは、現在の成果物構成と評価条件を簡潔にまとめたものです。

## フォルダ構成

```text
taiga_github_package/
├─ 2026_SolutionDeployment_Eval/   # 評価・可視化・投稿プラットフォーム
├─ taiga/                          # taiga の配置アルゴリズムと出力結果
└─ adv_hard_v2cmaes/               # 敵対的積み荷データ
```

## 入力と出力

標準入力:

```text
taiga_github_package/taiga/items_input.json
```

標準入力の出力:

```text
taiga_github_package/taiga/layout_result.json
```

敵対的データ:

```text
taiga_github_package/adv_hard_v2cmaes/hard_01.json ～ hard_12.json
```

敵対的データの出力:

```text
taiga_github_package/taiga/out_hard/layout_01.json ～ layout_12.json
```

## 主な制約

- コンテナサイズ: 12000(L) × 2300(W) × 2400(H) mm
- 最大積載重量: 24,000 kg
- 回転は水平方向の 0 度 / 90 度のみ
- コンテナ外にはみ出さない
- 積荷同士が重ならない
- 積荷が宙に浮かない
- 1 コンテナ内で配送先を混在させない
- Y 軸方向の重心が中心から ±3000 mm 以内

## 評価順

合格した結果は、以下の順で評価されます。

1. 使用コンテナ数が少ない
2. 平均 Y 軸重心ズレが小さい
3. 処理時間が短い
